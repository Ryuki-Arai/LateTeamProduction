## 制作前課題
* 制作における分業をどうするか
  * タスク/進捗の管理を各役職ごとに行う？
* ゲームをどの方向に広げるか
  * 2Dドットにすれば不自然さをぼかせそう？
* ステージ/ギミックに関して
  * マップ自動生成に関して
    * ステージギミックに何を伸ばすか
  * 枕返しのギミックに関して
    * 原案の中でもブレさせないで遊びに紐付ける

## 作業スケジュール
* 遊びの検証
  * 1部屋
  * 枕をひっくり返す機能を作る
  * １部屋に数人寝ていて枕返しのリザルト(True/False)パターンを実装する  
1. 枕を返すギミックの作成
2. 廊下の仲居のギミックの作成
3. マップの生成ギミックの作成

__5. マネージャー関係：__
